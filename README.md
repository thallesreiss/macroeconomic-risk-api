\# API de Risco de Crédito Macroeconômico (FastAPI)



Este projeto constrói uma API REST profissional e de alta performance utilizando FastAPI. Ele atua como a camada de serviço (Microserviço) para o modelo de Machine Learning treinado com dados oficiais do Banco Central do Brasil, permitindo o consumo de dados históricos e a previsão de risco de crédito em tempo real.



\## 🛠️ Arquitetura e Tecnologias

\- \*\*Framework Web:\*\* FastAPI e Uvicorn

\- \*\*Linguagem:\*\* Python 3.x

\- \*\*Banco de Dados:\*\* SQLite (Dados consolidados do BACEN)

\- \*\*Machine Learning:\*\* Scikit-Learn e Pandas

\- \*\*Governança:\*\* Git/GitHub
